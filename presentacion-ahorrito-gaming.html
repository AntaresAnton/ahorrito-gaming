Visión General del Proyecto

Ahorrito Gaming es una plataforma de e-commerce especializada en videojuegos que ofrece:

  • Análisis de datos para recomendaciones

Nuestro objetivo es proporcionar a los gamers una plataforma confiable donde puedan encontrar sus juegos favoritos a precios competitivos.

Arquitectura de Datos

Estructura de Base de Datos

El sistema está construido sobre una base de datos relacional MySQL con las siguientes características:

  • Más de 15 tablas interrelacionadas
  • Claves foráneas para mantener integridad referencial
  • Índices optimizados para consultas frecuentes
  • Soporte para transacciones seguras
  • Esquema escalable para futuras expansiones

Relaciones principales entre entidades

productos {
id: INT PRIMARY KEY
nombre: VARCHAR(255)
precio: DECIMAL(10,2)
RELATIONS: categorias, plataformas
}
usuarios {
id: INT PRIMARY KEY
email: VARCHAR(150)
RELATIONS: pedidos, favoritos
}
pedidos {
id: INT PRIMARY KEY
usuario_id: INT
total: DECIMAL(10,2)
FOREIGN KEY: usuarios(id)
}

Flujo de Compra

1. Navegación y Búsqueda

El usuario navega por categorías o utiliza el buscador para encontrar juegos de su interés.

Tablas: productos, categorias, plataformas

2. Selección de Productos

Agrega productos al carrito mientras continúa navegando por la tienda.

Tablas: carritos, productos

3. Proceso de Checkout

Revisa el carrito, aplica cupones y procede al pago.

Tablas: carritos, impuestos

4. Pago y Confirmación

Selecciona método de pago, completa la transacción y recibe confirmación.

Tablas: pedidos, metodos_pago

5. Entrega Digital

Recibe claves de activación o acceso a descargas en su perfil.

Tablas: pedidos, productos
